ColdNites

Grid-style turn-based strategy game(C++/UE4)

The game was built in Ue4 with the focus to make the level creation easy and modular. The game code extensively utilizes the inheritance and polymorphism behaviour of C++. The game architecture was planned using UML, process view and design documents with four people team.


I designed the high-level architecture for the game. Additionally, I designed and executed the following modules:

TileMap
Provides the Grid-based behaviour, and facilitates the event system. TileMap uses arrays and array maths, to store and spawn tiles and corresponding actors on the fly, convert world positions to tile coordinates, algorithms to get neighbour tiles, and many other helper functions.
